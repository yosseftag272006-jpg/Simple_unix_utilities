#Utilities (echo,cp,mv,pwd)

pwd : prints the path of the working directory.

cp : copies the content of a source file to a destination file.

mv : moves a file from a source path to a destination path.

echo : prints the user-input to the standard output (terminal).

#Example output

pwd

/home/usr/desktop

echo hello world

hello world

cp source.txt destination.txt

mv source.txt other_location.txt
